Category Title: name of the category, example: Fast and Furious

Description: describes how the words are related, example: Name car brands.

Words: words to add to the category with their point amount (the format is "newWord, number, newWord, number ..." and spaces do not matter),
example: "honda,1,mercedes,2, toyota, 3 , ford,100" so honda will give 1 point if you guess it, mercedes will give 2 points, ... and this is a valid input because spaces are overlooked.
